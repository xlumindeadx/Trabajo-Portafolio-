Tecnologia HTML5

Estructura la web (texto, imágenes, formularios)utilizado para estructurar y presentar contenido en la web .

Tecnologia CSS3

Da estilo y diseño (colores, fuentes, animaciones) se usa para dar estilo y presentación visual a los documentos HTML y XML.

Lenguaje de programación para desarrollo web.agrega interacción y dinamismo (botones, menús, validaciones, efectos).
